🚀 30-Second Setup

1 Open Notes app → create folder "Daily Log"
2 Open Shortcuts → tap +
3 Tap "Search actions" → add Date
(set format to: MMMM d, yyyy)
4 Add Create Note
Folder: Daily Log | Title: leave empty (it will use date)
5 Add Open Note
6 Tap ⋮ → name it "Daily Log" → Done!

✅ Now tap the button below - it should open your note!
