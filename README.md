# One-Tap Daily Log

A minimal, addictive daily journaling shortcut that builds a life archive with zero friction.

![Daily Log](https://img.shields.io/badge/Version-1.0-blue)
![Platform-iOS%20%7C%20macOS-purple)
![Build-Habit%20Daily-orange)

## Why This Shortcut?

Most journaling apps fail because they require too many taps, too much thinking, and too much setup. **Daily Log** is different:

- **One tap** to start
- **Zero setup** after initial configuration
- **Addictive** streak mechanics
- **Syncs everywhere** via iCloud Notes
- **Private** — your data stays yours

---

## Quick Start

### 1. Install the Shortcut

1. Open the **Shortcuts** app on your iPhone/iPad/Mac
2. Tap the **+** button → **Create Personal Automation**
3. Or download from: [Link to shortcut - see releases]
4. Configure your settings (see below)

### 2. First-Time Setup

```
Open Shortcuts → My Shortcuts → Daily Log
↓
Configure:
  • Log Folder Name: "Daily Log"
  • Enable Stats: Yes
  • Template Mode: On
```

### 3. Use It Daily

- **Morning**: Tap → See yesterday's summary → Add today's intentions
- **Evening**: Tap → Log wins → Review todos → Auto-save

## The Daily Log Format

Each entry follows this structure:

```
📅 [Date]
━━━━━━━━━━━━
🌅 MORNING

[Write your morning intentions here]

📊 YESTERDAY
────────────
Wins:
-

Todos:
-

📝 TONIGHT
──────────
Wins:
-

Todos for tomorrow:
-

💭 REFLECTION
─────────────
[Optional thoughts]
```

## Configuration Options

### Required
- **Notes Folder**: Name of the folder where logs are saved
- **Template Format**: How yesterday's content transfers

### Optional
- **Include Stats**: Show Reminders/ Calendar stats
- **Streak Tracking**: Count consecutive days
- **Time Reminders**: Morning + Evening notifications

## FAQ

**Q: Where is my data stored?**
A: In your personal iCloud Notes account. We don't store anything.

**Q: Can I export my logs?**
A: Yes! Open Notes → Select All → Share → Export as PDF/Markdown.

**Q: Does it work on Mac?**
A: Yes — Shortcuts app is built into macOS Monterey and later.

**Q: Can I customize the template?**
A: Absolutely. Edit the shortcut to change the format.
