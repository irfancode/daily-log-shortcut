## Installation

### Step 1: Create the Notes Folder
1. Open **Notes** app
2. Tap **Back** to see folders
3. Tap **New Folder** → Name it: `Daily Log`
4. Make sure **iCloud** is selected (not "On My iPhone")

### Step 2: Build the Shortcut

Since we can't directly share .shortcut files, follow these steps:

```
Shortcuts App → + (top right) → Create Shortcut
Name: "Daily Log"
```

**Add Actions (in order):**

1. **Text** → Enter today's date format:
   ```
   📅 [Today's Date]
   ```

2. **Get Notes** → Select folder "Daily Log", sort by Modified, limit 1

3. **If** result exists → Continue

4. **Get Text** from the note

5. **Text** → Template header:
   ```
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   🌅 MORNING

   -

   📊 YESTERDAY
   ──────────────────────────────
   ```

6. **Combine Text** → Join: Template + Yesterday's content

7. **Create Note** → In folder "Daily Log", title: Today's date

8. **Add to Note** → Combined content

9. **Open Note** → (No input - just show it)

### Step 3: Test It
- Run the shortcut once
- Check Notes → Daily Log → Should have today's entry
- Run again tomorrow — should auto-copy yesterday

## Daily Workflow

### Morning Routine (5 min)

```
1. ☕️ Coffee in hand
2. 📱 Tap "Daily Log"
3. ✍️ Fill in "Morning" section:
   - Today's top 3 priorities
   - How you're feeling
   - One intention for the day
4. ✅ Done - go live your day!
```

### Evening Routine (3 min)

```
1. 🏠 End of day
2. 📱 Tap "Daily Log" (or it's already open from morning)
3. ✍️ Fill in "Tonight" section:
   - 3 wins from today
   - 1 thing to improve
   - 2 todos for tomorrow
4. ✅ Close note - day logged
```

## Maximizing the Habit

### 🎯 The Streak Strategy

The magic of Daily Log is **consecutive days**. Here's how to protect your streak:

1. **Never miss two days in a row**
   - If you miss one, the next day is FREE — just do it
   - Gap penalty: It gets harder to restart

2. **Stack it with an existing habit**
   - After: Morning coffee
   - After: Evening brushing teeth
   - After: Checking email

3. **Use Back Tap (iPhone)**
   - Settings → Accessibility → Touch → Back Tap
   - Double tap → Daily Log
   - Now it's literally one tap (on the back!)

### 📊 The Stats Game

Track these metrics weekly:
- Days logged / Days in week (target: 7/7)
- Average words per entry
- Topics that repeat
- Win patterns

## Troubleshooting

### "Note not found"
- Verify "Daily Log" folder exists in Notes
- Check iCloud sync is enabled

### "Yesterday's content not showing"
- Make sure you logged yesterday first
- The shortcut copies the *most recent* note

### "Shortcut too slow"
- Reduce automation complexity
- Turn off "Show Stats" if not needed

### "Can't find the shortcut"
- Search in Shortcuts app
- Check "All Shortcuts" not "Gallery"
